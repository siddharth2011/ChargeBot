# Basic Chatbot using LangChain

A simple chatbot implementation using LangChain and OpenAI.

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Set your OpenAI API key:
```bash
export OPENAI_API_KEY='your-api-key-here'
```

## Usage

Run the chatbot:
```bash
python chatbot.py
```

Type your messages and press Enter to chat. Type 'quit' or 'exit' to end the conversation.

## Features

- Conversational memory - remembers context from previous messages
- Powered by GPT-3.5-turbo
- Simple command-line interface
